## 📌 Project Overview

This project was completed as part of the **CodeAlpha Data Analytics Internship**.

The objective is to analyze customer reviews and classify them into Positive, Negative, and Neutral sentiments using Natural Language Processing (NLP).

---

## 🎯 Objectives

* Clean text data
* Perform sentiment analysis
* Classify reviews
* Visualize sentiment distribution

---

## 🛠️ Tools & Libraries

* Python
* Pandas
* TextBlob
* Matplotlib
* Seaborn
* Regular Expressions (re)

---

## 📂 Dataset

Customer Reviews Dataset

Column:

* Review

---

## 📋 Steps Performed

1. Imported libraries
2. Loaded review dataset
3. Cleaned text using Regular Expressions
4. Calculated sentiment polarity using TextBlob
5. Classified reviews into Positive, Negative, and Neutral
6. Counted sentiment categories
7. Visualized results using Countplot

---

## 📈 Key Insights

* Most reviews are Positive.
* Negative reviews indicate areas for improvement.
* Overall customer sentiment is positive.

---

## 📸 Output

* Cleaned Reviews
* Sentiment Scores
* Sentiment Labels
* Countplot
